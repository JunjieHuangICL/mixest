Computes the cost function and the gradient at x in one call if possible.

function [cost, storedb] = getEuclideanCostGrad(problem, x, storedb)

Returns the value at x of the cost function described in the problem structure, as well as the gradient at x. The cache database storedb is passed along, possibly modified and returned in the process.
