agfactory

Construct an angular-Gaussian distribution structure

Syntax

D = agfactory(datadim)

Description

D = agfactory(datadim) returns a structure representing a datadim-dimensional angular-Gaussian distribution.

Distribution Parameters

  • sigma (datadim-by-datadim matrix)
